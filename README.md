
# React Tutorial Projects

A comprehensive collection of React learning projects covering fundamental to advanced concepts.

## Projects Overview

### 1. **React with JavaScript**
Basic React fundamentals using JavaScript.
- Components: State management, cards, video components
- Data handling with JS objects
- **Location**: `React_with_JavaScript/Demo-project`

### 2. **React with TypeScript**
Type-safe React development with TypeScript.
- Components: Alert, Button, ListGroup
- Type definitions and interfaces
- **Location**: `React_with_TypeScript`

### 3. **React Router Tutorial**
Client-side routing and navigation.
- Multi-page application structure
- Navigation bar and page routing
- **Location**: `React_router_tutorial/react-router-demo`

### 4. **Axios Demo**
HTTP client integration for API calls.
- API request handling
- Custom axios instance setup
- Backend integration
- **Location**: `Axios/axios-demo`

### 5. **Redux Tutorial**
State management with Redux.
- Redux store and slices
- Counter and science/social state management
- **Location**: `Redux_tutorial`

### 6. **To-Do List React**
Practical task management application.
- Task components and item management
- **Location**: `To_Do_list_React/to-do-list`

## Getting Started

Each project has its own `package.json`. Install dependencies:

```bash
cd <project-folder>
npm install
npm run dev
```

## Tech Stack

- React 18+
- Vite (build tool)
- TypeScript (optional)
- Redux Toolkit
- React Router
- Axios
